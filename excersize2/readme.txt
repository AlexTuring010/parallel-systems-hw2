Για να κάνεις compile το πρόγραμμα απλά πληκτρολογείς make.

Το εκτελέσιμο θα βρίσκεται μετά στο ./bin/main.

Το εκτελείς ως ./bin/main <matrix_dimension> <algorithm_type> <method_type> <num_of_threads>

Όπου:

algorithm_type: serial ή parallel

και

method_type: row ή column

Αν θέλεις, μπορείς να κάνεις export OMP_SCHEDULE="type, chunk_size"
για να καθορίσεις το scheduling που θέλεις να χρησιμοποιηθεί.

Στην έξοδο εμφανίζω μόνο τον χρόνο εκτέλεσης διότι στα tests που έκανα χρησιμοποιούσα πολύ μεγάλο μέγεθος πίνακα, οπότε δεν είχε νόημα να εκτυπώσω κάτι. Ωστόσο, στα αρχικά στάδια του προγράμματος έκανα πειράματα όπου επιβεβαίωσα ότι οι εξόδοι του είναι σωστοί.

Για τα tests μπορείς να τρέξεις πρώτα το ./scripts/collect_data.sh το οποίο θα τρέξει το πρόγραμμα για κάποια μεγέθη και κάποιους αριθμούς από νήματα. Τα αποτελέσματα αποθηκεύονται στο φάκελο data και στην συνέχεια το ./scripts/make_graphs.py μπορεί να τρέξει για να φτιάξει τα γραφήματα.
